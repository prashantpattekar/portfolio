School Result Management System

Back to My Portfolio
  • Developed a SCHOOL RESULTS MANAGEMENT SYSTEM project using Java, JSP, HTML, CSS , JavaScript and MySQL to manage and maintain student records and their academic results.
  • Designed a user-friendly interface consisting of two main pages, the student page, and admin page.
  • The student page allows students to view their academic results by entering their roll number
  • The admin page provides access to all student records and offers options to add new student details and results.
  • Created a normalized database schema to efficiently store and manage student records and their academic results.
  • Used MySQL to implement the database backend for the system, including creating tables, views, and stored procedures.
